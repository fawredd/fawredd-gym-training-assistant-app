# REQUIREMENT DOC: AI-001
Status: [APPROVED]
Role: Technical BA

## 1. Feature Description
Integration of the `AyudaMemoria` AI feature. Generates contextual recommendations for the next workout based on the user's recent history and goals.

## 2. Acceptance Criteria
- [ ] Function signature: `generateTrainingSuggestion(user, trainings, lastSuggestion) => string`.
- [ ] Collects the last 7 workouts of the user and their overarching goal/objective.
- [ ] Interacts with an LLM explicitly using `OpenRouter SDK` for Next.js configured with the `openrouter/free` model.
- [ ] Output must be structured text containing: a summary, a concrete recommendation, and suggested adjustments.
- [ ] Logs raw AI requests and responses to the DB for debugging purposes.
- [ ] Implements a Vercel Redis cache layer to store the last generated suggestion and limit generation frequency (e.g., max 1 per 12 hours) to avoid excessive API costs.
- [ ] The generation must NOT invent data outside the provided history.

## 3. Data Model Constraints
- `AiMemory`: Create a new table in Drizzle `fawredd_gym` schema: `ai_memories` with `id`, `user_id`, `fecha`, `contenido`.

## SECURITY REVIEW
[APPROVED]

- **Note:** Ensure AI outputs are sanitized on rendering (no raw HTML injection).
